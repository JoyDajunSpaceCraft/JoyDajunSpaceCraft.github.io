# Chapter 1  什么是JVM

+ 优点
1. 一次编写到处运行
2. 自动内存管理，垃圾回收

## JVM 学习路线
java class -> java classLoader <--> 内存结构(方法区 -> 堆 -> 虚拟机栈 -> 程序计数器 -> 本地方法栈) <--> 执行引擎(解释器 -> 即时编译器 -> GC 垃圾回收)

学习顺序是1. 内存结构 2. 垃圾回收 3. java class 4. java classLoader 5. 即时编译器
